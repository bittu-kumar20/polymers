# Modo Polymers PLC

A modern and responsive corporate website for **Modo Polymers PLC**, a reliable plastic injection molding manufacturer based in Ethiopia. The website showcases the company's products, manufacturing capabilities, and business information with a clean and user-friendly interface.

## 🌐 Live Demo

https://polymers-plc.vercel.app/

## 🚀 Features

- Responsive design for all devices
- Modern UI built with Tailwind CSS
- Product showcase section
- About company section
- Contact form with EmailJS integration
- Smooth scroll animations using AOS
- SEO optimization with React Helmet
- Fast page loading with Vite
- Client-side routing using React Router

## 🛠️ Tech Stack

- React.js
- TypeScript
- Tailwind CSS
- React Router DOM
- Vite
- EmailJS
- AOS (Animate On Scroll)
- React Helmet Async
- React Hot Toast
- React Icons
- Tabler Icons

## 📦 Installation

Clone the repository

```bash
git clone  https://github.com/bittu-kumar20/polymers.git
```

Go to the project directory

```bash
cd modo-polymer
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

## 🎯 Purpose

This project was developed to provide a professional online presence for **Modo Polymers PLC**, helping customers explore the company's plastic manufacturing services, products, and contact information through a fast, responsive, and modern web application.
